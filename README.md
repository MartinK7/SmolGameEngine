# SmolGameEngine
Unfinished game engine with some limitations.

- Cubemap shadows
- Mirror effects
- Textures

## Screenshot of SGE_Demo
![Screenshot of SGE_Demo](demo/assets/screenshots/Screenshot_from_2022-09-13_18-15-33.png)

## How to build
```console
mkdir build
cd build 
cmake .. 
cmake --build .
./SGE_Demo
```
