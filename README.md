# SmolGameEngine
Unfinished game engine with some limitations.

- Cubemap shadows
- Mirror effects
- Textures

## Screenshot of SGE_Demo
![Screenshot of SGE_Demo](demo/assets/screenshots/Screenshot from 2022-09-13 18-15-33.png)

## How to build
```console
mkdir build
cd build 
cmake .. 
cmake --build .
./SGE_Demo
```
